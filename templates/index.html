Mengapa Ada 3 Hasil Model ML?

Sistem skrining tugas akhir ini dikembangkan dengan membandingkan tiga algoritma Machine Learning berbeda dari dataset medis CDC. Hal ini bertujuan untuk menganalisis akurasi terbaik:

GB

Gradient Boosting (Model Utama)

Akurasi 84.52%. Merupakan model terbaik yang dideploy secara penuh sebagai acuan medis utama karena performanya yang unggul dalam mendeteksi pola penyakit secara non-linear. Status: Aktif sebagai Sistem Deteksi Utama

RF

Random Forest (Model Pembanding)

Akurasi 83.20%. Algoritma ensembel bagging untuk mengurangi overfitting dan menjaga stabilitas keputusan prediksi.

LR

Logistic Regression (Model Pembanding)

Akurasi 81.88%. Pendekatan statistik parametrik linier klasik yang digunakan sebagai garis dasar (baseline) pembanding efisiensi komputasi.
